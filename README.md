# [Workout Logger](http://youreweakbro.herokuapp.com)

Simple app to log workouts! Built using Ruby on Rails and React / Redux.

### Design Documents
- [Schema](./docs/schema.md)
- [Wireframes](./docs/wireframe.md)

### Stack
- [Ruby on Rails](http://rubyonrails.org)
- [React](https://facebook.github.io/react/)
- [Redux](https://github.com/rackt/redux)
- [ImmutableJS](https://facebook.github.io/immutable-js/)
- [Webpack](https://webpack.github.io/)
- [Sass](http://sass-lang.com/)
- Plotly or D3

### Features
##### Key Features
- [ ] Sign in and sign up
- [ ] Add exercises
- [ ] Log workouts

##### Additional Features
- [ ] Log meals
- [ ] Log weight
- [ ] Log injury
- [ ] Plan workouts
- [ ] Graph workouts
- [ ] Follow users
- [ ] Like workouts
- [ ] Tag workouts and exercises e.g. 'Olympic Lifting' or 'Bodybuilding'

### Setup
1. Run `bundle install`
2. Run `npm install`
3. Run `webpack`
4. Run `rails s`
5. Go to `localhost:3000` on the browser
